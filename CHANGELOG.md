- **config.py**: Добавлена константа `BACKTEST_DEFAULT_BARS`

### ✨ Новые функции

#### performance_monitor.py (НОВЫЙ МОДУЛЬ)
- Мониторинг времени обработки баров
- Отслеживание использования памяти и CPU
- Измерение задержек API запросов
- Подсчёт ошибок
- Контекстный менеджер `TimingContext` для удобного измерения

#### risk_calculator.py (НОВЫЙ МОДУЛЬ)
- Kelly Criterion - оптимальный размер позиции
- Value at Risk (VaR) - максимальный ожидаемый убыток
- Conditional VaR (CVaR) - средний убыток в худших случаях
- Risk of Ruin - вероятность разорения
- Sharpe Ratio - доходность с учётом риска
- Sortino Ratio - доходность с учётом downside риска
- Calmar Ratio - доходность / максимальная просадка
- Класс `RiskAnalyzer` для комплексного анализа

#### analyze_risks.py (НОВЫЙ СКРИПТ)
- Быстрый анализ рисков из командной строки
- Автоматические рекомендации по RISK_PCT
- Оценка качества торговой системы
- Расчёт оптимального размера позиции

#### analyze_risks.bat (НОВЫЙ ФАЙЛ)
- Удобный запуск анализа рисков в Windows

## [2.0] - 2026-04-07

### ✨ Основные функции
- Мультитаймфреймовый анализ (5m/15m/1h/4h)
- Бэктестинг на реальных данных Bybit
- Динамическое управление позицией по волатильности
- Защита от просадки (drawdown protection)
- Управление портфелем с контролем корреляции
- Комиссии биржи учтены во всех расчётах
- Расширенный дашборд (6 вкладок)
- Асинхронная архитектура для 30 пар
- Отдельная модель для каждой пары

### 📁 Структура
- 30 обученных моделей CatBoost
- Логирование сигналов, сделок, китов
- WebSocket для real-time данных
- Rate limiting для API
