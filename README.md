# 🚀 Bybit CatBoost Trading Bot v2.1

Профессиональный торговый бот для Bybit Futures с машинным обучением (CatBoost), мультитаймфреймовым анализом и продвинутым риск-менеджментом.

## ✨ Основные возможности

### 🤖 Машинное обучение
- **30 отдельных моделей CatBoost** - по одной для каждой торговой пары
- **40+ технических индикаторов** - EMA, RSI, MACD, Bollinger Bands, ADX, Stochastic, OBV, CMF, VWAP
- **Автоматическое переобучение** - модели обновляются каждые 500 баров
- **Валидация качества** - новая модель принимается только при улучшении F1 score

### 📊 Мультитаймфреймовый анализ
- Анализ сигналов на **4 таймфреймах**: 5m, 15m, 1h, 4h
- Определение **лучшего ТФ** для входа по уровню уверенности
- Расчёт **TP/SL/R:R** для каждого таймфрейма
- Подсчёт **confluence** - согласованности сигналов

### 💼 Управление портфелем
- **Контроль корреляции** между парами (макс. 3 коррелированных позиции)
- **Лимиты экспозиции**: 80% общий, 25% на пару
- **Динамический sizing** - размер позиции масштабируется по волатильности
- **Защита от просадки** - остановка торговли при DD > 8%

### 🐳 Smart Money детекция
- **Whale bars** - обнаружение крупных игроков (объём > 3× среднего)
- **Absorption** - поглощение спроса/предложения
- **Stop Hunt** - охота за стоп-лоссами

### 🔬 Бэктестинг
- Тестирование на **реальных исторических данных** Bybit
- Метрики: **Win Rate, PnL, Sharpe, Profit Factor, Max Drawdown**
- **Equity curves** для визуализации
- Поддержка тестирования отдельных пар или всего портфеля

### 📈 Дашборд (Streamlit)
**6 вкладок:**
1. **Сканер** - текущие сигналы по 30 парам с фильтрами
2. **Сделки** - история, PnL, equity curve, статистика
3. **Киты** - активность smart money
4. **AI Модель** - статус, feature importance
5. **Мультитаймфрейм** - анализ по ТФ для каждой пары
6. **Бэктест** - запуск тестов с визуализацией

### ⚡ Производительность
- **Асинхронная архитектура** - параллельная обработка 30 пар
- **WebSocket** для real-time данных (вместо REST polling)
- **Rate limiting** - защита от превышения лимитов API
- **Мониторинг** - отслеживание CPU, памяти, задержек API

### 🛡️ Риск-менеджмент
- **Kelly Criterion** - оптимальный размер позиции
- **VaR / CVaR** - оценка максимального убытка
- **Risk of Ruin** - вероятность разорения
- **Sharpe / Sortino / Calmar** - коэффициенты качества системы
- **Trailing Stop Loss** - защита прибыли
- **Комиссии биржи** учтены во всех расчётах (0.11% round-trip)

## 📦 Установка

### 1. Требования
- **Python 3.12+**
- **Windows / Linux / MacOS**
- **Bybit аккаунт** (testnet или mainnet)

### 2. Установка зависимостей

```bash
cd bybit-catboost-bot-v2
pip install -r requirements.txt
```

### 3. Настройка API ключей

Отредактируйте `config.py`:

```python
API_KEY    = "ваш_api_key"
API_SECRET = "ваш_api_secret"
TESTNET    = True  # False для реальной торговли
```


В отдельном терминале:

## ⚙️ Конфигурация

Основные параметры в `config.py`:

### Торговля
```python
SYMBOLS = [...]  # 30 ликвидных пар
RISK_PCT = 0.01  # 1% капитала на сделку
LEVERAGE = 5
MAX_POSITIONS = 1  # на пару
```

### Риск-менеджмент
```python
SL_ATR_MULT = 1.5  # Stop Loss = 1.5 × ATR
TP_ATR_MULT = 2.5  # Take Profit = 2.5 × ATR
TRAIL_ACTIVATE_ATR = 1.0  # Trailing SL после 1 × ATR прибыли
MAX_DRAWDOWN_PCT = 0.08  # Остановка при DD > 8%
```

### Портфель
```python
MAX_TOTAL_EXPOSURE = 0.80  # Макс. 80% капитала в позициях
MAX_EXPOSURE_PER_PAIR = 0.25  # Макс. 25% на одну пару
MAX_CORRELATED_PAIRS = 3  # Макс. коррелированных позиций
CORRELATION_THRESHOLD = 0.70  # Порог корреляции
```

### Модель
```python
LONG_PROB_THRESH = 0.62  # Порог для LONG сигнала
SHORT_PROB_THRESH = 0.62  # Порог для SHORT сигнала
RETRAIN_EVERY_N = 500  # Переобучение каждые N баров
```

### Комиссии
```python
TAKER_FEE = 0.00055  # 0.055%
MAKER_FEE = 0.00020  # 0.020%
ROUND_TRIP_FEE = 0.0011  # 0.11% (вход + выход)
```

## 🎯 Рекомендации

### Для начинающих
1. Начните с **TESTNET** (`TESTNET = True`)
2. Используйте **консервативные настройки**: `RISK_PCT = 0.005` (0.5%)
3. Запустите **бэктест** перед реальной торговлей
4. Следите за **дашбордом** первые дни

### Для опытных
1. Оптимизируйте `RISK_PCT` через **Kelly Criterion**
2. Настройте **корреляционные лимиты** под свой портфель
3. Экспериментируйте с **порогами вероятности** (0.60-0.65)
4. Используйте **мониторинг производительности**

### Переход на реальную торговлю
1. Накопите **минимум 100 сделок** на testnet
2. Проверьте **Win Rate > 50%** и **Sharpe > 1.0**
3. Убедитесь, что **Risk of Ruin < 5%**
4. Измените `TESTNET = False` в `config.py`
5. Используйте **mainnet API ключи**
6. Начните с **малого капитала**

## 🐛 Решение проблем

### Бот не запускается
- Проверьте Python версию: `python --version` (нужен 3.12+)
- Установите зависимости: `pip install -r requirements.txt`
- Проверьте API ключи в `config.py`

### Модели не обучены
- Запустите: `python pretrain_all.py`
- Проверьте папку `models/pairs/` (должно быть 60 файлов)

### Высокая задержка API
- Проверьте интернет-соединение
- Используйте VPN если Bybit заблокирован
- Увеличьте `MIN_PAIR_SPACING_SEC`

### Много ошибок в логах
- Проверьте лимиты API на Bybit
- Уменьшите количество пар в `SYMBOLS`
- Увеличьте `API_RATE_LIMIT`
